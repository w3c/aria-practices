# WAI-ARIA APG Skill

Copy this folder to `.cursor/skills/` in your project so the AI uses WAI-ARIA Authoring Practices when building accessible components.

## Setup

```sh
# From your project root
mkdir -p .cursor/skills
cp -r /path/to/aria-practices/skill/wai-aria-apg .cursor/skills/
```

Add the APG MCP server to Cursor (Settings → MCP) so the AI can fetch pattern specs.
