ARIA Example: Color Picker Using Slider Widgets

This example implements the slider design pattern that is often found in color pickers. Change the sliders to update the background color of box (note: the box is visually below the 3 slider controls), with the HEX and RGB values in form controls.

ARIA Roles, Properties and States

Role Attribute Element Usage
slider div Identify the widget as a slider
aria-valuemax div Maximum value of the slider
aria-valuemin div Minimum value of the slider
aria-valuenow div Current value of the slider
aria-labelledby div Reference to define a unique descriptive accessible name for each slider widget
