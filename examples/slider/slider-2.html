ARIA Example: Vertical Sider and Sliders using aria-valuetext

This example implements the slider design pattern for a vertical slider and for sliders using aria-value-text in a hypothetical online thermostat. Change vertical slider to set a value for temperature. The fan speed and heat/air conditioning sliders use the aria-value-text property to communicate a text value, rather than a numerical value to assistive technologies.

ARIA Roles, Properties and States

Role Attribute Element Usage
slider div Identify the widget as a slider
aria-orientation div Indicates the orientation of the slider is "vertical" for the temperature slider or "horizontal" for the fan and temperature controls
aria-valuemax div Maximum value of the slider
aria-valuemin div Minimum value of the slider
aria-valuenow div Current value of the slider
aria-valuetext div The value of the temperature slider appended with the text "degrees" or one of the text values
aria-labelledby div Reference to define a unique descriptive accessible name for each slider widget
