Keyboard Support

Carousel Tabs

Key Function
Right Arrow
  • Moves focus to the next tabpanel (e.g.).
  • If last tabpanel is focused, focus moves to the first tabpanel.
Left Arrow
  • Moves focus to the previous tabpanel.
  • If first tabpanel is focused, focus moves to the last tabpanel.

Navigation Buttons

Key Function
Space
Enter		 Moves focus to corresponding tabpanel (e.g. slide).
Left button moves to previous tabpanel. Right button moves to next tabpanel.

Start/Pause Button

Key Function
Space
Enter		 Button is clicked.
If carousel is running, button pauses it. If carousel is paused, carousel will restart.

Role, Property, State, and Tabindex Attributes

Role Attribute Element Usage
tablist div Identifies a list of tabs. Each tab controls the display of the slide.
tab li
  • Indicates which tabpanel (e.g. slide) in the carousel is visible.
  • When focused, it automatically displays the corresponding tabpanel (e.g. slide).
aria-selected=true li aria-selected=true indicates that the tabpanel (e.g. slide) associated with the tab is currently visble.
aria-selected=false li aria-selected=false indicates that the tabpanel (e.g. slide) associated with the tab is not currently visble.
aria-labelledby=[IDREF] li
  • aria-labelledby=[IDREF] provides an accessible name for the tab.
  • [IDREF] references the id attribute of the h2 element in the corresponding tabpanel.
aria-controls=[IDREF] li
  • aria-controls=[IDREF] references the id attribute of the corresponding tabpanel.
  • The aria-controls provides a programmatic reference for the scripting to hide and show each tabpanel.
  • The aria-controls relationship provides a way for assisitive technologies to provide enhanced navigation.
tabindex=0 li
  • The tab that is selected.
  • See roving tabindex for more information.
tabindex=-1 li
  • All the tabs except the one that is selected.
  • See roving tabindex for more information.
tabpanel div
  • Indicates the element serves as a container for a tab panel.
  • Is hidden unless its associated tab control is activated.
aria-label a, div
  • aria-label defines the string element that labels the current element.
  • aria-label provides the user with a recognizable name for the current element.
aria-labelledby div
  • The purpose of aria-labelledby is the same as that of aria-label.
button a
  • These buttons are actually links, though in this case, do not refer to anywhere on this page or elsewhere.
  • Generally, links redirect focus elsewhere. In this case, however, focus remains on the button when clicked.
