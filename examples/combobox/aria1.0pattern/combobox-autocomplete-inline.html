Example of Legacy ARIA 1.0 Combobox With Inline Autocomplete

NOTE: This page is ready to start reviewing. This work is tracked by issue 99.

The below example section demonstrates a combobox that implements the ARIA 1.0 design pattern for combobox. As the user types in content an autocomplete option will be presented in the textbox if a match is found in a list of options. The user only get to see the matches, there is no listbox to see the entire list of options.

Similar examples include:

  • Combobox Without Autocomplete : example demonstrates the behavior associated with aria-autocomplete=none.
  • Combobox With an List Autocomplete : example demonstrates the behavior associated with aria-autocomplete=list.
  • Combobox With Both List and Inline Autocompletion : example demonstrates the behavior associated with aria-autocomplete=both.
