Textbox Keyboard Support

Key Function
Down Arrow Opens listbox and moves focus to first option.
Alt+Down Arrow Opens listbox and focus stays on the input[type="text"].
Up Arrow Opens listbox and moves focus to last option.
Alt+Up Arrow Opens listbox and focus stays on the input[type="text"].
Escape if open, closes the listbox.

Listbox Keyboard Support

Key Function
Enter
  • input[type="text"] value is set to the option content.
  • Closes the listbox.
  • Sets focus on the input[type="text"]
Escape
  • Closes the listbox.
  • Sets focus to the input[type="text"] button.
Up Arrow
  • Moves focus to the previous option.
  • If focus is on the first option, the focus does not move.
Down Arrow
  • Moves focus to the next option.
  • If focus is on the last option, the focus does not move.
Home
Page Up		 Moves focus to the first option.
End
Page Down		 Moves focus to the last option.
A-Z
a-z
  • Moves focus to the option with a label that starts with the typed character if such an option exists.
  • Otherwise, focus does not move.
