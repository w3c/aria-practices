ARIA Roles, Properties and States

Role Attribute Element Usage
button div, a
  • role="button" identifies the element as a button widget.
  • Accessible name for the button is defined by the text content of the element.
tabindex="0" div, a
  • Makes the element keyboard focusable and part of tab order of the page.
  • The a element needs to define tabindex=0 to become part of the tab order on the page, when it is missing or has an empty href attribute.
aria-pressed="false" a
  • Identify a element with the toggle button behavior as toggled off
aria-pressed="true" a
  • Identify a element with the toggle button behavior as toggled on
