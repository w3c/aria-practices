Pizza Ordering Form with Popup Errors

NOTE: This is an early draft of a new pattern. Please provide feedback in issue 252.

The following pizza ordering form example using standard HTML5 form controls demonstrates how to make required form fields, identifying forms with invalid values, instructional information and popup error messages accessible to screen reader users using the Form Input Assistance design pattern.

HTML5 Accessibility Features

  • Form input[type=text] and select controls are labeled using the label[for=IDREF] reference technique.
  • Form input[type=radio], input[type=checkbox] and output controls are labeled using the label encapsulation technique.
  • The HTML5 required attribute is used to indicate required form controls.
  • The HTML5 disabled attribute is used to indicate disabled form controls.
