This document provides readers with an index of ARIA Authoring Practices 1.1 examples using WAI-ARIA 1.1 [[WAI-ARIA]] to create accessible rich internet applications. The index provides references to examples by role, properties and states. Not every role, property and state may have an example.

This document is part of the WAI-ARIA suite described in the WAI-ARIA Overview.
